Instructions
Authorization
  1. To authorize, please create an app here ( https://accounts.salesloft.com/oauth/applications/new ) with this Redirect URI (<%= params.partnerData.callbackUrl %>). For the name, please enter “LeanData” along with the name of your company. This naming convention assists SalesLoft with troubleshooting any issues.
  2. SalesLoft will then provide a Client ID and Client Secret.
  3. Return to this page, populate the Application ID (Client ID) and Client Secret, and submit the form to authorize LeanData access to SalesLoft.
Implementation Steps
This Self-Serve Guide walks through each step in detail.
  1. Enter the Retry Action time interval. This time interval determines when we will retry initiating an action in SalesLoft after a failure.
    • An example of failure is when LeanData cannot find the CRM ID on the SalesLoft record; hence, we recommend aligning this interval to your Salesforce-SalesLoft Sync settings.
    • You can change this value to as small as 1 minute.
    • Note that there is a limit to the amount of retries we will initiate. Once that limit is reached, the action will fail indefinitely.
  2. Add Node(s) to the Graph
  3. Review Edge options
